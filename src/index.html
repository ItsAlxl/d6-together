Privacy Policy

d6 Together is open-source, meaning the source code is publicly available. Users who are inclined to run their own instance of d6 Together are encouraged to do so.

All data that d6 Together receives has to be explicitly entered by the user, and none of it is personally identifiable unless the user intentionally enters personally identifiable information (don't!). The information is transmitted user-to-user via the server, but the server never stores information.
